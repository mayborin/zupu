# 族谱 - 家族传承

一个交互式家族族谱创建与可视化工具。

## 功能

- **录入信息**：以家庭为单位录入父亲、母亲、子女的姓名、性别、出生年份
- **自动去重**：同名同出生年份自动识别为同一人
- **族谱可视化**：自动生成多代族谱图，夫妻合并卡片展示
- **图上编辑**：点击节点可编辑、添加配偶/子女、删除
- **导入导出**：支持 JSON 格式导入导出族谱数据
- **智能布局**：子女自动对齐父母下方，关系线不交叉

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 技术栈

- React 18
- Vite
- SVG 可视化（纯手写，无第三方图表库）

## 使用说明

1. 打开后点击「＋ 添加一个家庭」开始录入
2. 填写父亲、母亲姓名和出生年份，添加子女
3. 重复添加更多家庭，跨家庭的同一人会自动合并
4. 点击「🌳 生成族谱图」查看可视化
5. 在族谱图上点击节点进行编辑操作
6. 点击「📤 导出JSON」保存数据
